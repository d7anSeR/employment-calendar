# Календарь занятости сотрудников 📅
**Концепция продукта:**

Продукт представляет собой интуитивно понятный и функциональный календарь занятости сотрудников, который интегрируется с системой учета рабочего времени, разработанной на базе 1С и размещенной на веб-сервере. Календарь поможет эффективно управлять рабочим временем и загрузкой сотрудников, обеспечивая прозрачность и удобство в планировании задач.

**Название команды:**
Дамский квартет +1

**Группа:** 6512-100503D

## Блок работы с API
* Добавление заявки в БД
**POST** /api/webhook/schedule
```
{
  "employeeName": "Иван",
  "startDate": "2025-09-28T09:00:00",
  "endDate": "2025-09-28T10:00:00",
  "numberTask": 7654,
  "viewTask": "личная"
}
```
* Получение информации о заявке по сотруднику
**GET** /api/webhook/user/Кирилл
* Получение информации о заявке по дате
**GET** /api/webhook/date/2025-09-28
* Получение информации о заявке по сотруднику и дате
**GET** /api/webhook/user/Кирилл/date/2025-09-28
